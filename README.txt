Topshop - E-Commerce Website

1. Extract the 'topshop' folder to 'XAMPP > htdocs' or 'WAMP > www'.
2. Go to 'XAMPP > htdocs > topshop > functions > send-email.php'.
3. Modify lines 23, 24 and 25 with valid credentials of a Gmail account. 
4. Open XAMPP or WAMP Control Panel.
5. Start Apache and MySQL.
6. Open a web browser and go to PHPMyAdmin at 'http://localhost/phpmyadmin'.
7. Create a new database named 'topshop'.
8. Import the SQL file 'XAMPP > htdocs > topshop > topshop.sql' to the new database.
9. Open a web browser and browse Topshop at 'http://localhost/topshop'.
